Budget Shield: Personal Finance Tracker
Budget Shield is a full-stack financial management application designed to track expenditures with precision and reliability. This project serves as a demonstration of Data Persistence, Relational Database Management, and Server-Side Rendering.

✨ Core Functionalities
Persistent Storage: Uses a SQLite database to ensure user data is saved and remains available after browser restarts or server reloads.

Full CRUD Logic: Demonstrates the ability to Create, Read, Update, and Delete data entries within a relational database.

Real-Time Aggregation: Automated Python logic calculates total spending dynamically as new entries are added.

Responsive UX: A sleek, dark-mode fintech interface built using Tailwind CSS for a modern user experience.

🛠️ Technical Architecture
Backend: Python 3.10+, Flask (Web Framework)

Database: SQLAlchemy ORM (Object Relational Mapper)

Frontend: Jinja2 Templates, HTML5, Tailwind CSS

Environment: Isolated virtual environments for dependency management.
